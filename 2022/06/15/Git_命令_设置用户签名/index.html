Git_命令_设置用户签名

git安装，看视频：B站尚硅谷

git安装好的第一件事就是设置用户签名
安装好git之后，需要进行设置一下用户签名，否则无法提交代码。
签名的作用是区分不同操作者身份。用户的签名信息在每一个版本的提交信息中能够看到，用来确认本次提交是谁做的。
注意：这里设置用户签名和将来登录GitHub（或者其他的托管中心）的账号是没有任何关系的。

  1. 设置用户名：git config --global user.name 用户名

  2. 设置邮箱：git config --global user.email 邮箱
